# MMO
Это многопользовательская браузерная игра, над которой я работаю в свободное время. Проект находится в стадии активной разработки.

## Стек технологий
* **Python 3.10+**
* **FastAPI**
* **SQLAlchemy 2.0 + aiosqlite** (Асинхронная ORM)
* **vkbottle** (Интеграция с VK LongPoll)
* **Алгоритмы:** Perlin Noise (генерация карты), A* (поиск пути)

## Что уже реализовано:
* Асинхронный движок на FastAPI с разделением на роутеры
* Реляционная БД
* Генерация континента с биомами и прокладка дорог между городами
* Пошаговая боевая система с расчетом инициативы и статусами
* Интеграция с VK

## В планах:
* Доработка экономики (аукцион)
* Настройка деплоя
* Миграция с SQLite на PostgreSQL
